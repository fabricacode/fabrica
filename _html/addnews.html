Adding a new news post is a simple process. Just follow the guidelines listed below and your post will appear on the front page of Fabrica's site.

Title
This is the main header that is displayed both on the front page and in large text at the top of the news post page.

Subtitle
This is the subheader that appears below the title only on the news post page. It will not appear on the front page

Link
This is the clean URL where the post can be accessed. For example, if we have the title "Fabrica Goes to Salone", then we would want to enter something like "fabrica-goes-to-salone" (without quotes), so that the post would appear at http://www.fabrica.it/news/fabrica-goes-to-salone.

Body Text
This is the main content of the news post. Feel free to use HTML and paste in video embedding code to make it appear on the news post page.

Main Image
This is the image that will appear on the front page as well as at the top of the news post. A thumbnail will be automatically generated. JPG is preferred.

Tags
Tags used to describe the content of the post. Seperate them with a comma such as "design, exhibition, installation".
